TRIAL

The title of the type 2 project I have done is “The Enemy Returns” on account of the comic made on the story ‘The Enemy’ written by Pearl S. Buck. This project entails a detailed report about it. Furthermore, this project focuses on the message passed on by pearl s. buck. Another goal would be to boost academic interest in the chapter by giving a visual illustration that is by the comic. This project mainly contains the process I went through while making the comic such as character analysis, scene analysis, etc. this project is sort of a reminder that chapters we study in schools can also be made a source entertainment and fun if we put our minds into it.
